91 //! Determine if the EEPROM is ready for communication by writing
92 //! the address+!write byte and looking for an acknowledge. This is
93 //! repeated every 1ms until an acknowledge occurs, or a timeout occurs.
94 //! If a timeout occurs, an I2C stop is generated.
95 //! Returns 0 if an acknowledge is generated and 1 if not.
96 int8_t eeprom_poll(uint8_t i2c_address);
