Detailed Description

DISABLE_INPUT_UVCL Bit Field.

Disables automatic power reduction in response to low input voltage.

  • Register: REG0
  • CommandCode: 0x00
  • Size: 1
  • Offset: 7
  • MSB: 7
  • MASK: 0x80
  • Access: R/W
  • Default: 0
