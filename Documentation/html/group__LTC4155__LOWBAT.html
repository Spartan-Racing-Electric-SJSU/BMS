Detailed Description

LOWBAT Bit Field.

Battery below ~2.8V or charger disabled.

  • Register: REG3
  • CommandCode: 0x03
  • Size: 1
  • Offset: 0
  • MSB: 0
  • MASK: 0x01
  • Access: R
  • Default: n/a
