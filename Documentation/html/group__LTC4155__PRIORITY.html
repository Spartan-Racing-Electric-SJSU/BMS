Detailed Description

PRIORITY Bit Field.

Input connector priority swap setting. Setting PRIORITY will cause power to be drawn from USB when both WALL and USB power are present.

  • Register: REG1
  • CommandCode: 0x01
  • Size: 1
  • Offset: 7
  • MSB: 7
  • MASK: 0x80
  • Access: R/W
  • Default: 0
