Detailed Description

REG0 Register.

7 6 5 4:0
DISABLE_INPUT_UVCL EN_BAT_CONDITIONER LOCKOUT_ID_PIN USBILIM
  • CommandCode: 0x00
  • Contains Bit Fields:
    • DISABLE_INPUT_UVCL : Disables automatic power reduction in response to low input voltage.
    • EN_BAT_CONDITIONER : Enables autonomous Overtemperature Battery Conditioner Load when the thermistor temperature exceeds 60Â°C.
    • LOCKOUT_ID_PIN : Disables autonomous USB OTG operation when the ID pin is grounded by an appropriate USB cable.
    • USBILIM : USB total input current limit setting. Register contents will be modified by the LTC4155 upon certain events. See the datasheet section Input Current Regulation.
